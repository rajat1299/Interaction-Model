# Fitted-v1 calibration evidence heads

The 417 materialized browser bundles and runtime databases were removed after Phase 1 close. `calibration-manifest.json` preserves their deterministic identities and SHA-256 digests; the retained source package and input profile preserve the small regeneration inputs.

`report-adjudicated-v2.json` is the final historical fitted-v1 report and preserves the accepted engineering finding. Its legacy evidence-admission fields are superseded by the project-owner closeout decision and are not an active framework. `SHA256SUMS` binds only the retained files in this directory.
